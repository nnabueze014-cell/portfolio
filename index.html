Project 2 – SLA-Based Support Ticket Automation

Project Goal: Design an automated support system that enforces SLA compliance, escalates breaches, and improves operational visibility.

How it was achieved:

  1. Configured Freshdesk to manage ticket intake and SLA timers
  2. Automated priority escalation when first response or resolution SLA was breached
  3. Reassigned escalated tickets and notified supervisors via Slack
  4. Synchronized tickets with ClickUp tasks for operational tracking
  5. Updated task status and added tagged comments when due dates were reached

Key Outcomes:

  • Improved SLA compliance and escalation visibility
  • Clear ownership of overdue and high-priority tickets
  • Reduced response delays through automated alerts

Tools Used: Freshdesk, Zapier, ClickUp, Slack
