Project 2 – Support Ticket Automation System (In Progress)

Project Goal: Design and deploy a centralized technical support system that automatically converts customer support requests into actionable tasks, assigns them to the correct technical pipeline, and enforces resolution timelines.

System Overview:

  1. Customers submit issues through a Freshdesk web widget.
  2. Tickets are categorized by service type (Solar, Electrical, HVAC, CCTV, Smart Home).
  3. Webhooks push ticket data into Zapier.
  4. Zapier creates structured tasks in ClickUp.
  5. Priority levels control automatic due-date assignment.
  6. SLA monitoring ensures overdue tasks are flagged for escalation.
  7. Slack notifications keep teams informed in real time.

Key Features:

  • Automatic task creation from support tickets
  • Priority-based due date assignment
  • Service-specific pipelines
  • SLA tracking and escalation readiness
  • Timezone-aware task handling

Tools Used: Freshdesk, Zapier (Webhooks), ClickUp, Slack.

Status: Actively implemented and tested for internal company use. SLA optimization and escalation logic ongoing.
