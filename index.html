Project 1 – Lead Automation

Project Goal: Automate lead collection, follow-up, and internal notifications to reduce manual effort and ensure timely engagement.

How it was achieved:

  1. Google Form collects incoming leads.
  2. Responses stored in Google Sheets.
  3. Zapier triggers automation workflows.
  4. Automatic email response via Gmail.
  5. Task creation in HubSpot CRM.
  6. Follow-up task in Asana.
  7. Slack notification to the team.

Key Outcomes:

  • Reduced manual processing
  • Improved response time
  • Clear task ownership
  • Consistent follow-up workflow

Tools Used: Google Forms, Google Sheets, Zapier, Gmail, HubSpot, Slack, Asana.

Project 2 – Support Ticket Automation System

Project Goal: Build a centralized support system that converts customer issues into actionable tasks with SLA tracking.

How it was achieved:

  1. Support requests submitted via Freshdesk widget.
  2. Tickets categorized by service and priority.
  3. Freshdesk webhooks send data to Zapier.
  4. Zapier creates tasks in ClickUp.
  5. Automatic due dates based on priority.
  6. Slack notifications provide visibility.

Key Outcomes:

  • Automated ticket-to-task conversion
  • Clear SLA ownership
  • Improved response accountability
  • Reduced operational overhead

Tools Used: Freshdesk, Zapier (Webhooks), ClickUp, Slack.
